# 구현 계획: 한양대 수강신청 AI Agent

## Overview

React + TypeScript 프론트엔드, AWS Lambda (Node.js 18) 백엔드, DynamoDB + OpenSearch Serverless 데이터 계층, Amazon Bedrock AI 모듈로 구성된 서버리스 수강신청 AI Agent를 구현한다. 각 태스크는 P1(필수/핵심), P2(중요), P3(선택/개선) 우선순위로 분류되며, 의존 관계가 있는 경우 명시한다.
